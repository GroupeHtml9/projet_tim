Biographie du fondateur de HTML

Tim Berners-Lee: biographie:

Sir Timothy John Berners-Lee est né le 8 juin 1955 à Londres. Ses parents étaient tous deux mathématiciens et ont travaillé sur l’un des premiers ordinateurs commerciaux, ce qui l’a très tôt plongé dans le monde de l’informatique.
Il étudie la physique à l’Université d’Oxford, où il se passionne pour les systèmes informatiques et les réseaux. Après ses études, il travaille comme ingénieur logiciel, notamment au CERN (Organisation européenne pour la recherche nucléaire) en Suisse.

Approche historique:

La naissance du HTML et du Web

En 1989, alors qu’il est au CERN, Tim Berners-Lee propose un projet visant à faciliter le partage d’informations entre chercheurs. Il imagine alors un système reposant sur :

  • le HTML (HyperText Markup Language) pour structurer les documents;
  • le HTTP pour les échanges;
  • et les URL pour identifier les ressourses.

En 1991, le premier site web est mis en ligne. Le HTML devient rapidement le langage de base du Web, permettant de créer et de relier des pages entre elles grâce aux liens hypertextes.
